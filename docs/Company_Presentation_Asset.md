# Company Presentation Template
**Brand:** Shetue Filling Station / Shetue MultiBiz
**Status:** Generic Professional Layout (Navy blue + teal + light gray)
**Fonts:** Montserrat + Open Sans

## Access Link
[Open Google Slides Template](https://docs.google.com/presentation/d/1ZiBmKgkz3FjV9wYJvPb7pW6DQ5Xe4Zr2/edit?usp=sharing)

## Instructions to Edit
1. Click **File > Make a copy…**
2. Name it: `Sheetoefillingstation_Presentation`
3. Click **OK** — now it’s yours to customize!

## Slide Guide
| Slide | What to Do |
|------|-----------|
| **Slide 1** | Replace subtitle if needed. Add your name/team if presenting. |
| **Slide 2–3** | Keep as-is—they’re generic and powerful. |
| **Slide 4** | All 5 businesses are already listed—just verify names. |
| **Slide 6** | Stats show “$X+”, “X years”—just type real numbers when ready. |
| **Slide 8** | Fill in your phone/email when available. Leave placeholders if not. |

*You can edit the branding colors later via Slide > Theme Builder.*
